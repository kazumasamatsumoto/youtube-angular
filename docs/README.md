# 動画制作ドキュメント

このディレクトリには、各動画の台本とシナリオコンテが含まれています。

## 📁 構成

```
docs/
├── README.md                    # このファイル
└── 01-no-onpush-app/           # 第1回動画
    ├── 台本.md                  # ナレーション台本
    └── シナリオコンテ.md        # シーン構成と撮影指示
```

## 📝 ドキュメントの種類

### 台本（台本.md）
- ナレーションの全文
- セリフとタイミング
- 画面構成の説明
- 補足メモ

### シナリオコンテ（シナリオコンテ.md）
- シーンごとの詳細な構成
- 時間配分
- 画面表示内容
- ナレーション内容
- 撮影時の注意点
- BGM・効果音の指示

## 🎬 動画制作の流れ

1. **企画・構成** - 企画案.mdを参照
2. **台本作成** - このディレクトリに台本を作成
3. **シナリオコンテ作成** - 撮影用の詳細な指示を作成
4. **実装** - サンプルアプリを作成（examples/）
5. **撮影** - シナリオコンテに従って撮影
6. **編集** - 台本とシナリオコンテを参照して編集
7. **公開** - Webサイトに詳細解説を追加
